# JS-Monster-Slayer-Game
Monster Slayer Game in Vue.js

- Built on Vue v3 

### Methods 

- startGame()
- attackMonster()
- attackPlayer()
- specialAttackMonster()
- healPlayer()
- surrender()
- addLogMessage(who, what, value) This method has three parameters for the battle log.
